# 📝 Segunda Práctica de Lenguajes de Programación (Python) 
![Python](https://img.shields.io/badge/Python-3.x-blue)

Este repositorio contiene la **segunda práctica** de la materia Lenguajes de Programación, con ejercicios de Python organizados por temas: Strings, Recorridos, Funciones, Listas, Tuplas, Diccionarios, Sets y ejercicios integradores.

## 🔧 Contenido de los ejercicios

### 1️⃣ Ejercicios (Strings)
1. Pedir una palabra y mostrarla en mayúsculas, minúsculas y *title case*.  
2. Pedir una frase y contar cuántas veces aparece cada vocal.  
3. Dada una frase, mostrar las 3 primeras y las 3 últimas letras usando *slicing*.  
4. Verificar si una palabra **contiene** la letra `'r'`.

### 2️⃣ Ejercicios (Recorridos)
5. Pedir una palabra y contar cuántas vocales tiene.  
6. Ingresar 4 palabras y mostrar únicamente las que contengan la letra `'r'`.  
7. Ingresar palabras hasta escribir `FIN`; imprimir las que **empiecen y terminen** con la misma letra.

### 3️⃣ Ejercicios (Funciones)
8. `es_palindromo(cadena)`: devuelve `True` si la cadena se lee igual al derecho y al revés (ignorar espacios y mayúsculas/minúsculas).  
9. `contar_vocales(cadena)`: retorna un diccionario con la cuenta de cada vocal.  
10. `normalizar_palabras(frase)`: retorna una **lista** de palabras sin signos de puntuación y en minúsculas.

### 4️⃣ Ejercicios (Listas)
11. Ingresar notas hasta `-1`, calcular el **promedio** y cuántos están **por debajo**.  
12. Dada una lista de palabras, construir otra con las que tengan **más de 5 letras**.

### 5️⃣ Ejercicios (Tuplas)
13. Guardar coordenadas `(x, y)` en una tupla e imprimir la distancia al origen.  
14. Dada una lista de tuplas `(nombre, edad)`, imprimir solo los mayores de 18.

### 6️⃣ Ejercicios (Diccionarios)
15. Leer nombres y notas hasta `FIN` y guardar en un diccionario. Luego mostrar:  
   - Promedio general  
   - Mejor estudiante  
   - Estudiantes por debajo del promedio  
16. Construir un diccionario `{palabra: longitud}` a partir de una frase.

### 7️⃣ Ejercicios (Sets)
17. Pedir dos listas de números (separados por espacios) y mostrar su **intersección** usando `set`.  
18. Eliminar duplicados de una lista manteniendo el **orden** (pista: usar un `set` auxiliar para vistos).

### 8️⃣ Ejercicios Integradores
19. `frecuencias_letras(frase)`: devuelve un **diccionario** con la cantidad de apariciones de cada letra (ignorar espacios y may/min).  
20. Programa de calificaciones: leer `(nombre, nota)` hasta `FIN` y luego mostrar:  
   - Promedio  
   - Mediana  
   - Mejor y peor nota  
   - Informe ordenado por nombre  
21. Dada una frase, construir el set de **letras únicas** y mostrar cuántas son; además, construir una **lista ordenada** de esas letras.

## Instrucciones

```bash
git clone https://github.com/tu_usuario/Segunda-Practica.git
```
2. Abrir los archivos `.py` en VS Code o tu editor preferido.  

3. Ejecutar los ejercicios en Python 3.
